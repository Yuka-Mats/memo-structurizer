## 推奨ビジネスモデル（ピラミッド構造）

```
         ▲
        / \
       /   \  コンサル（濃い人向け・少数）
      /  △  \  ← 月額コミュニティのトップ層
     /-------\
    /         \  コミュニティ（継続収益）
   /    ○      \  ← ハンズオンからの導線
  /-----------\
 /             \  ハンズオンイベント（集客軸）
/      ◎        \  ← エントリーポイント
```

**導線設計:**
1. **ハンズオンイベント**で顧客獲得（無料体験活用？）
2. **コミュニティ**への導線設計（濃い人ほど参加）
3. **コンサル**で単価を上げる（コミュニティ内の信頼から）

## 実装の懸念点

### 1. キャパシティ
- 現在のリソース体制では全部は無理
- 段階的なロールアウトが必須
- 優先順位の厳格な管理

### 2. コミュニティ運営
- モデレーター不可欠（自分1人では無理）
- 雰囲気づくり・メンバー管理が大変
- 質問への回答SLA設定が重要

### 3. 初期集客
- 無料体験（ハンズオン）で集客する？
- 既存顧客の紹介ループ構築

### 4. 価格設定
- コミュニティ月額：**3,000円 / 5,000円 / 8,000円？**
- コンサル：**月額いくら？**
- ツールキット：**一回買い切り？**

### 5. ブランド名
- 「◯◯ラボ」：実験感
- 「◯◯部」：コミュニティ感
- 他候補？
