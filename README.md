# Memo Structurizer

🗂️ **自動的に雑なメモを整理して、構造化されたMarkdownファイルに変換するツール**

## 概要

テキスト形式の散らかったメモを入力すると、内容を自動判定して複数の整理されたMarkdownファイルに分割・保存します。

**対応するメモ種類:**
- 📋 **会議** - ミーティングの議事録
- 📚 **学習** - 学習ノート、技術情報
- 🎤 **ヒアリング** - インタビュー、顧客フィードバック
- 💡 **アイデア** - ブレインストーミング、提案

## インストール

### 前提条件
- Python 3.7 以上

### セットアップ

```bash
# リポジトリをクローン
git clone https://github.com/Yuka-Mats/memo-structurizer.git
cd memo-structurizer

# セットアップスクリプトを実行（初回のみ）
python setup.py
```

セットアップスクリプトが以下を行います:
- `~/Documents/my-knowledge/` ディレクトリを作成
- 日付ごとのサブディレクトリ構造を作成

## 使い方

### 方法 1: コマンドラインから実行

```bash
python memo_structurizer.py "メモ種類" "メモの内容"
```

**例:**

```bash
# 会議メモを整理
python memo_structurizer.py "会議" "今日の会議で以下のことが決まった..."

# 学習メモを整理
python memo_structurizer.py "学習" "Pythonのデコレータについて学んだ..."

# ヒアリングメモを整理
python memo_structurizer.py "ヒアリング" "顧客から以下の要望があった..."

# アイデアメモを整理
python memo_structurizer.py "アイデア" "新しい機能として以下を検討中..."
```

### 方法 2: 対話的に使用

メモ種類を指定しない場合、自動判定または質問されます:

```bash
python memo_structurizer.py "メモの内容がここに入ります..."
```

メモ種類が不明な場合:
```
❓ メモの種類が分かりません。
以下から選んでください：
  1. 会議
  2. 学習
  3. ヒアリング
  4. アイデア

選択 (1-4): 
```

### 方法 3: Python コードから使用

```python
from memo_structurizer import MemoStructurizer

# インスタンスを作成
structurizer = MemoStructurizer()

# メモを処理
memo_content = "今日の会議で決まったことは..."
saved_files = structurizer.process_memo(
    memo_content, 
    memo_type="meeting",
    title="2026年5月定期会議"
)

# 作成されたファイルを確認
for file_path in saved_files:
    print(f"保存されました: {file_path}")
```

## ファイル構造

メモを処理すると、以下のディレクトリ構造で保存されます:

```
~/Documents/my-knowledge/
├── 2026-05-22/
│   ├── 01-会議-概要.md
│   ├── 01-学習-概要.md
│   ├── 01-ヒアリング-概要.md
│   └── 01-アイデア-概要.md
├── 2026-05-23/
│   └── ...
```

各ファイルは、メモ種類に最適なテンプレート構造で作成されます。

### 会議メモのテンプレート
- アジェンダ
- 議論内容
- 決定事項
- アクション アイテム
- フォローアップ

### 学習メモのテンプレート
- 主要概念
- 主な学び
- 詳細説明
- 質問・確認点
- 参考資料

### ヒアリングメモのテンプレート
- 主要な発見
- 詳細なインサイト（Q&A形式）
- アクション アイテム
- フォローアップ質問

### アイデアメモのテンプレート
- コアアイデア
- 実装時の検討事項
- 必要なリソース
- 関連アイデア

## メモ種類の自動判定

以下のキーワードを含むメモは自動判定されます:

| 種類 | キーワード |
|------|----------|
| 会議 | 会議, meeting, ミーティング, 議論, 決定, アクション, 参加者 |
| 学習 | 学習, learning, 学ぶ, 勉強, 概念, 理解, 参考資料 |
| ヒアリング | ヒアリング, interview, インタビュー, 質問, 回答, 発見 |
| アイデア | アイデア, idea, ブレイン, 提案, 検討, 実装, 機能 |

**注意:** メモ種類が不明な場合、自動的にユーザーに質問します。決して推測しません。

## 設定

デフォルト保存ディレクトリを変更したい場合:

```python
from memo_structurizer import MemoStructurizer

# カスタムディレクトリを指定
structurizer = MemoStructurizer(base_dir="/path/to/your/directory")
```

## トラブルシューティング

### ディレクトリが作成されない
- `~/Documents/` ディレクトリが存在することを確認してください
- Windows の場合、パスが正しく認識されているか確認してください

### ファイルが文字化けしている
- エディタで UTF-8 エンコーディングで開いてください

### メモ種類が自動判定されない
- キーワードが含まれていない可能性があります
- 手動でメモ種類を指定してください
